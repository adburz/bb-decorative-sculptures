# Fixtures

Lokalne dane dla dev mode (`PUBLIC_CONTENT_SOURCE=fixtures`). Typy zgodne z docelowym schema Sanity.

Kiedy Faza 4 wdroży adapter Sanity, fixtures zostają jako fallback / testowy zestaw danych. Nie usuwać.
